ALEX - A Linguistic EXchange geobot

ALEX is a natural language interface for geographical data, such as openstreetmap data. It performs natural language processing to analyze the intent of the user, after which it generates SQL queries and responds with the corresponding GeoJSON, which is displayed on the screen. Keep in mind that it is still in very early development.
ALEX is fully opensource and available on github.
